# Easycom Commands

    AZ             - (No Parameters) Read azimuth
    AZ [xx]x[.xxx] - Initiate azimuth rotation to a heading	
    EL		Elevation
    ML		Move Left (CCW)
    MR		Move Right (CW)
    MU		Move Up
    MD		Move Down
    SA		Stop azimuth rotation
    SE          Stop elevation rotation
